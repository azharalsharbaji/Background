Background Image,Let's Go!

Properties

There are few different background Properties:

background-image

This property is location of image itself ,for example
h1 { background-imge: url('https://website.com/img/background.jpg');}
Or
h1 { background-imge: url('img/background.jpg');}

background-color

This allows us to add colour to a background. an example in CSS would like this:
h1 { background-color: blue;}

background-repeat

This is how the image will repeat. The value con be any of the following:
- background-repeat: repeat;
This repeats on both the X and Y axis , and will essentially tile the image across the entire background.
- background-repeat: repeat-x;
This repeats the background on X axis ,or horizontally.
- background-repeat: repeat-y;
This repeats the background on Y axis ,or horizontally.
- background-repeat: no-repeat;
This shows the image only once, without repeating.

background-position

This is the actual position of the background image within it's container.the values can be:

background-position: top;

background-position: right;

background-position: bottom;

background-position: left;

background-position: center;

it can also be a percentage, length,or combination,like so:
background-position: 50px 20%;

Shorthand!

There's a quick way around background imager, and you don't necessarily have to type out all of the above code to set up your background images the way you want them.

Background

The "background" Shorthand can handle them all in one!
background: blue url('img/background.jpg') top left no-repeat;
